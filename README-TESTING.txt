# Bourbon Pourcast User Testing Static Site

This is a no-risk frontend prototype. It does not overwrite your existing homepage unless you rename `bp-test-index.html` to `index.html`.

## Files

- `bp-test-index.html` — open this directly for testing
- `bp-test.css` — mobile-first styling
- `bp-test.js` — filtering and UI behavior
- `bp-test-data.js` — static test data

## Suggested testing location

Put these files in your `bourbon-pourcast-frontend` repo root and visit:

`/bp-test-index.html`

Example:
`https://bourbonpourcast.com/bp-test-index.html`

## Notes

The alert form is UI-only. It does not store emails or send alerts yet.
